# Polytechnic and University Management System

A comprehensive Flutter application for managing polytechnic and university operations.

## Features
- Student Login
- Teacher Login
- Attendance Tracking
- Results Management
- Class Routines
- Notices
- Assignment Upload

## Tech Stack
- Flutter
- Dart

## Setup
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
