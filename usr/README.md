# Polytechnic and University Management System

A comprehensive cross-platform Flutter application for managing university and polytechnic operations, featuring dedicated panels for Students, Teachers, and Administrators.

## Features

*   **Role-based Authentication**: Secure login and registration for Students, Teachers, and Admins.
*   **Student Panel**: View attendance, results, routines, assignments, notices, subjects, semester fees, library status, and exam schedules. Includes a modern dashboard with quick access and daily updates.
*   **Teacher Panel**: Take attendance, upload results and assignments, view class routines, manage student lists, and upload notices.
*   **Admin Panel**: Comprehensive management of students, teachers, courses, departments, results, and overall analytics.
*   **Real-time Ready**: Structured to seamlessly integrate with Firebase (collections pre-defined for users, students, teachers, attendance, results, etc.) or Supabase.

## Getting Started

1.  Ensure you have Flutter installed.
2.  Run `flutter pub get` to fetch dependencies.
3.  Run `flutter run` to launch the application.
